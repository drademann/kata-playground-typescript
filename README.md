# kata-playground-typescript

[![Wallaby.js](https://img.shields.io/badge/wallaby.js-configured-green.svg)](https://wallabyjs.com)
[![code style: prettier](https://img.shields.io/badge/code_style-prettier-ff69b4.svg?style=flat-square)](https://github.com/prettier/prettier)

This is a project setup to quickly get started with various Coding Katas using TypeScript.

Currently the following Katas are included:

- FizzBuzz
- Prime Factors (thanks to _Uncle Bob_ [Robert C. Martin's Prime Factor Kata](http://butunclebob.com/ArticleS.UncleBob.ThePrimeFactorsKata))
- Roman Number Converter
- String Calculator (thanks to [Roy Osherove](http://osherove.com/tdd-kata-1/))

A description with requirements for each Kata is included in each test.

_Hint_: You may use Wallaby's feature of taking `.only` into account to only execute the tests of your current Kata.

Remember the TDD and Kata rules:

- Do one task at a time.
- Do it incrementally.
- Solve requirements as simple as possible.
- Refactor after each successful test.
- This is about learning TDD/BDD _"red, green, repeat"_
